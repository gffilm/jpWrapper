What Would You Do?
Sue Lin has licensed software on her PC that another associate wants to copy. What should she do?

A. Let the associate copy the software.

B. Give the software to the associate but suggest that they should get a license.

If you answered B, you are correct. The software is licensed and the associate will need to order the software under an additional license so that it can be used legally.
